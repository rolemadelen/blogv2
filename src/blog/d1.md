폼에서 input, button, textarea, 등등 클릭시 푸른 테두리가 생긴다.

![blue outline](/images/blue-outline.webp)

CSS를 이용해서 테두리를 없앨수있다.

```css
input:focus {
    outline: none;
}
```

다른 스타일에 의해 `outline`이 덮어 쓰인다면 `!important`로 해당 스타일을 강제할 수 있다.

```css
input:focus {
    outline: none !important;
}
```
